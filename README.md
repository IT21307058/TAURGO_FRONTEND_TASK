# Todo App - Modern Task Manager

A beautiful, feature-rich todo list application built with React and Tailwind CSS.

## Features

- Add, edit, and delete todos
- Mark todos as completed
- Filter by all/pending/completed
- Dark mode support
- Fully responsive design
- LocalStorage persistence
- Beautiful, colorful UI
- Smooth animations
- Drag-and-drop to reorder

## Installation
```bash
# Clone the repository
git clone [https://github.com/yourusername/todo-app.git](https://github.com/IT21307058/TAURGO_FRONTEND_TASK)

# Navigate to project directory
cd todo-app

# Install dependencies
npm install

# Start development server
npm start
```

## Tech Stack

- React 
- Tailwind CSS
- Lucide React (icons)
- LocalStorage API
